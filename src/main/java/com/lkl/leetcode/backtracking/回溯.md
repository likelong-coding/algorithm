**回溯算法模板**

结果可以定义为全局变量
```java
void backtracking(参数) {
    if (终止条件) {
        存放结果;
        return;
    }

    // for循环遍历树的宽度，剪枝一般也是从宽度入手，还有就是题目一些已知条件也可以用作剪枝
    for (选择：本层集合中元素（树中节点孩子的数量就是集合的大小）) {
        处理节点;
        backtracking(路径，选择列表); // 递归，树的深度
        递归之后，回溯，撤销处理结果
    }
}
```

回溯算法：**模板 + 剪枝**
    最好是能画出对应多叉树型结构，便于理解和剪枝
    ；回溯算法的实质就是穷举搜索，剪枝可以
